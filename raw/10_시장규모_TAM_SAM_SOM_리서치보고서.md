## 1. 전체 시장 구조와 정의

이 시장은 단일한 하나의 시장이 아니라 **4개의 층위가 중첩된 복합 시장**이다. 각 층위를 이해해야 시장 규모를 올바르게 산정할 수 있다.

```
[글로벌 지능형 자동화 시장]  ← Layer 1: 글로벌 TAM
        ↓
[국내 AI·RPA·자동화 소프트웨어+서비스 시장]  ← Layer 2: 국내 TAM
        ↓
[국내 중소/중견 제조업 특화 세그먼트]  ← Layer 3: SAM
        ↓
[정부 지원금 연계 + 초기 공략 가능한 구체적 고객군]  ← Layer 4: SOM
```

**시장 정의 (본 보고서 기준)**

| 포함 범위 | 제외 범위 |
|----------|----------|
| AI 기반 업무자동화(RPA·사무OA) 컨설팅 | 순수 하드웨어(설비·PLC) 공급 |
| 에이전트 AI·LLM 기반 솔루션 구축 SI | 대기업(500인 이상) 대상 SI |
| 스마트공장 고도화 소프트웨어 레이어 | 자체 보유 AI 모델 개발(파운데이션) |
| 정부 바우처 연계 컨설팅 패키지 | 단순 소프트웨어 라이선스 재판매 |

## 3. SAM: 확보 가능 세부 시장

### 3-1. 중소/중견 제조업 AI 자동화 특화 세그먼트

**Bottom-up 추정 근거**

| 항목 | 수치 | 근거 |
|------|------|------|
| 국내 중소/중견 제조기업 총 수 | **163,273개사** | 중소벤처기업부, 2025년 스마트제조혁신 실태조사 (2023년 기준) |
| 스마트공장 도입 기업 비율 | **19.5%** (약 31,838개사) | 중기부 제1차 스마트제조혁신 실태조사 |
| 그 중 기초 수준에 머무는 비율 | **75.5%** → 고도화 잠재 수요 24,038개사 | 동 실태조사 |
| 제조 AI 도입 기업 비율 | **0.1%** (약 163개사) | 동 실태조사 |
| 스마트공장 미도입 기업 | 약 **131,435개사** → 신규 진입 잠재 수요 | 계산값 |
| 스마트공장 평균 도입 비용 | 약 **7억 5,000만 원** | 중기부 실태조사 |

**규모별 AI 도입률 (실증 데이터)**

| 기업 규모 | 스마트공장 도입률 | 해석 |
|----------|----------------|------|
| 중견기업 (300명 이상) | **85.7%** | 이미 높은 수준, 고도화·AI 레이어 추가 수요 |
| 중기업 (50~299명) | **54.2%** | 기초 도입 완료 → AI 업그레이드 수요 폭증 예정 |
| 소기업 (10~49명) | **28.5%** | 신규 진입 수요 가장 큼, 단가 낮음 |
| 소상공인 (1~9명) | **8.7%** | 도입 초기 단계, 바우처 없이는 진입 어려움 |

**SAM 추정치 (RPA + AI 컨설팅 서비스 포함 기준)**

```
국내 RPA 시장 전체 (소프트웨어+서비스): 약 3,000~4,500억 원 추정
  × 중소/중견 제조업 비중 (약 20~30%): → 약 600~1,350억 원

정부 지원 유발 수요 포함 시 상한:
  → 약 800~1,500억 원 (2024년 기준)

2027년 성장 전망:
  → 약 1,600~3,000억 원
```

> **계산 근거**: RPA 소프트웨어 시장(1,500억) × 서비스 배수(2~3x) = 3,000~4,500억. 여기서 제조업 중소기업 비중 20~30% 적용.

## 4. SOM: 실제 목표 시장

### 초기 3년 공략 가능한 현실적 시장 규모

**Top-down 접근**

```
SAM (중소/중견 제조업 AI 자동화 서비스): 약 800~1,500억 원
  × 신규 진입자 현실적 점유 가능 비율 (약 1~5%):
  → SOM: 약 8~75억 원 (연간)
```

**Bottom-up 접근 (3년 로드맵 기준)**

| 단계 | 목표 고객 수 | 평균 계약 단가 | 연 매출 추정 |
|------|------------|-------------|------------|
| 1년차: 중소 제조업 문서자동화 | 5~10개사 | 500~1,500만 원 | 2,500만~1.5억 원 |
| 2년차: 중견 제조업 ERP 연동 | 3~5개사 | 3,000만~1억 원 | 9,000만~5억 원 |
| 3년차: 스마트팩토리 고도화 | 2~3개사 | 5,000만~3억 원 | 1억~9억 원 |
| **합산 (3년 누적)** | **10~18개사** | - | **약 12.5억~15.5억 원** |

## 5. 정부 지원 예산이 만드는 인공 수요

> 이 시장의 핵심 특수성: **정부가 수요를 인위적으로 창출한다.** 바우처 없이는 구매하지 않을 기업이, 바우처 때문에 구매한다.

### 5-1. 주요 지원 프로그램별 예산 (2024~2026년)

| 프로그램 | 주관 부처 | 예산 규모 | 기업당 지원 한도 | 비고 |
|---------|---------|---------|--------------|------|
| **AI 바우처** (2024년) | NIPA (과기부) | 201개 과제 선정 | 최대 **2억 원** | 공급·수요 컨소시엄 방식 |

| **산업·제조 AX 사업** (2026년) | 중기부/과기부 합산 | **4,230억 원** | 사업별 상이 | AI 에이전트·스마트공장 고도화 통합 |

### 5-2. 연간 정부 지원 총액 추정

```
중기부 전체 예산 (2025년): 약 15.2조 원
  ↓ 이 중 디지털전환·제조혁신 관련 추정: 약 5,000억~1조 원
  
AI 바우처 + 혁신바우처 + AX 사업 합산 (2026년 기준):
→ 약 5,000억~5,500억 원 이상

단순 환산: 이 예산이 시장에 풀리면
→ 연간 최소 5,000개 이상 기업이 AI/자동화 솔루션 도입에 정부 지원금 활용
```

## 6. 고객 유형별 행위 단계 분석

> **핵심 인사이트**: 고객은 AI 자동화 서비스를 '한 번에 구매'하지 않는다. 인식→탐색→검토→결정→도입→확산의 6단계를 거치며, 각 단계마다 다른 개입이 필요하다.

### 6-1. 고객 유형 매트릭스

```
                    [AI 이해도 높음]
                         │
            박과장        │      최부장
         (중견 IT담당) ───┼─── (공장장/TF장)
                         │
[예산 작음] ─────────────┼──────────────── [예산 큼]
                         │
            김원장        │      이사장님
          (소상공인) ────┼─── (중소 제조 대표)
                         │
                    [AI 이해도 낮음]
```

### 6-2. 고객 유형별 행위 단계 분석 (6단계 퍼널)

#### 🏭 타입 A: 중소 제조업 대표 "이사장님" (종업원 30~150명)

| 단계 | 행위 | 트리거 | 개입 포인트 |
|------|------|--------|-----------|
| **1. 인식** | 구인난 체감, 지인 사례 청취 | 직원 퇴사, 업계 세미나 | 업종 협회 세미나 발표, 사례집 배포 |
| **2. 탐색** | 정부 지원금 검색, 지인 소개 요청 | 바우처 공고 노출, 주변 소개 | 바우처 연계 자료, 1페이지 사례 요약지 |
| **3. 검토** | 비용 견적 요청, 레퍼런스 확인 | "우리 같은 회사도 됩니까?" | 무료 현장 진단, 동종업 레퍼런스 공유 |
| **4. 결정** | 정부 지원 연계 조건 확인 | 지원금 있으면 YES, 없으면 WAIT | 바우처 신청 대행, ROI 계산서 제시 |
| **5. 도입** | PoC → 전사 확산 | 초기 1~2개 업무 자동화 성공 체험 | 30일 무상 지원, 사용 교육 영상 |
| **6. 확산** | 옆 공장 소개, 추가 계약 | 절감 효과 수치 확인 | 성과 보고서 제공, 고도화 제안 |

**핵심 구매 행동 데이터**:
- 구매 기준 우선순위: ① 비용(정부 지원 여부) → ② 효과 가시화 → ③ 지원/AS 여부
- 구매 저항 원인: "직원들이 못 쓸 것 같다"(40%), "사기 같다"(30%), "비쌀 것 같다"(30%) (추정)
- 의사결정 기간: 바우처 연계 시 평균 2~4주, 미연계 시 3~9개월

#### 🖥️ 타입 B: 중견기업 IT 담당자 "박과장" (종업원 300~500명)

## 7. 온라인 컨설팅 지불의사(WTP) 근거 데이터

### 7-1. AI 구독 서비스 지불 실증 데이터

| 지표 | 수치 | 출처 |

### 7-2. 세그먼트별 추정 WTP (지불의사)

| 고객 유형 | 추정 WTP | 현실적 지출 구조 | 민감도 |
|----------|---------|--------------|-------|
| **중소 제조 대표 (이사장님)** | 연 500~1,500만 원 | 정부 지원금으로 800만 원 + 자부담 200만 원 | 정부 지원 여부에 극도로 민감 |
| **중견 IT 담당 (박과장)** | 연 3,000만~1억 원 | 예산 승인 후 선투자 가능 | ERP 연동 여부에 민감 |
| **소상공인 (김원장)** | 월 3~10만 원 | 카드 자동 결제 구독 | 즉각적 편의성에 민감 |
| **스마트팩토리 TF (최부장)** | 연 5,000만~3억 원 | 정부 고도화 지원 연계 전제 | 기술 검증 선행 필요 |

### 7-3. WTP를 높이는 조건

아래 세 가지 조건이 충족될 때, 구매 결정 속도가 3~5배 빨라진다는 것이 현장 관찰 결과다:

1. **수치화된 ROI**: "월 80시간 절감 = 인건비 약 160만 원 상당" 형태로 제시
2. **정부 지원금 연계**: "본인 부담 200만 원에 나머지 800만 원은 정부에서" 구조
3. **비슷한 회사의 레퍼런스**: "반월시화 △△금속 도입 후 3개월 만에 경리 시간 70% 감소"

### 7-4. 구독형 모델의 중소기업 수용성

```
중소기업의 AI/디지털 서비스 구독 트렌드:
 ✅ "디지털 월세" 문화 정착 중 (초기 구축 비용 부담 큰 CapEx → 월정액 OpEx 선호)
 ✅ 즉각적인 업무 효율화가 가능한 영역(문서요약, 고객응대, 마케팅 자동화)부터 AI 도입
 ✅ 정부 바우처와 연계된 구독 서비스 수요 증가 (민관 협력 플랫폼 활성화)
 ⚠️ 소상공인은 성과 체감 전 WTP가 매우 낮음 → "무료→성과→유료 전환" 퍼널 필수
```

## 📌 최종 요약: 한 페이지 시장 스냅샷

```
┌─────────────────────────────────────────────────────────────────┐
│              시장 규모 스냅샷 (2024년 기준)                        │
├─────────────────────────────────────────────────────────────────┤
│ 글로벌 TAM (지능형 자동화)    │ 약 145~166억 달러 (약 20~23조 원) │
│ 국내 TAM (AI 산업 전체)       │ 약 6조 3,000억 원               │
│ 국내 TAM (RPA+서비스 포함)    │ 약 3,000~4,500억 원 (추정)       │
│ 국내 SAM (제조 중소기업 특화)  │ 약 600~1,500억 원               │
│ 정부 인위 수요 (연간 바우처 등) │ 약 5,000억 원 이상              │
│ SOM (초기 3년 목표)           │ 약 8~75억 원 (점유율 1~5%)        │
├─────────────────────────────────────────────────────────────────┤
│ 스마트공장 도입 기업 중 AI 미도입   │ 약 31,675개사 → 즉시 공략 가능 │
│ AI 도입률 (제조업)                 │ 0.1% → 성장 잠재력 극대        │
│ 소상공인 디지털 활용 증가 속도     │ 연 +9.2%p → 수요 확대 중       │
│ AI 구독 기업 중 월 100만↑ 지출     │ 45% → WTP 존재 확인           │
└─────────────────────────────────────────────────────────────────┘
```

**핵심 결론 3가지**:

1. **시장은 크다, 하지만 내 몫은 작은 조각이다**: 전체 시장은 조 단위이지만, 신규 진입자가 3년 내 현실적으로 가져갈 수 있는 규모는 10~75억 원 수준이다. 이 안에서 이익률을 높이는 것이 관건.

2. **정부가 수요를 만든다**: 이 시장의 수요 약 40~60%는 정부 바우처에 의해 인위적으로 창출된다. 바우처 공급사 등록 없이는 영업이 사실상 불가능하다.

3. **고객의 WTP는 조건부다**: 소상공인은 무료가 아니면 쓰지 않고, 중소 제조 대표는 정부 지원 없으면 미루고, 중견 IT 담당자는 ERP 연동이 안 되면 검토조차 않는다. WTP를 높이는 건 기술이 아니라 **"조건의 충족"** 이다.

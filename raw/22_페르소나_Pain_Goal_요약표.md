| **Core-03** | 정미경 (CIO, 45세) | SEG-C | ERP·MES 데이터가 파편화되어 전혀 연동 안 됨. 클린 데이터 구축에 수개월이 걸려 시작 못함 | 시스템 교체 없는 '데이터 통합 브릿지' 완성. 실시간 전사 가시성 확보 |
| **Core-04** | 소장님 (공장장, 55세) | SEG-C/D | 숙련 작업자 은퇴로 기술 전수 단절. 현장 반장들의 데이터 입력 거부감이 극심함 | 작업자 입력 없이 AI가 공정 로그를 읽어 일일 생산 보고서를 자동 생성 |
| **Core-05** | 박준호 (AX TF장, 41세) | SEG-D | 임직원 간 AX 온도 차이 극심. PoC 실패 시 본인 커리어에 직접 타격 | 3개월 내 Quick-Win 수치 창출로 대형 AX 예산 확보 |
| **Adjacent-01** | 이재무 (CFO, 49세) | 확장 | 각 부서 IT 예산안이 '좋아질 것'이라는 장밋빛 기대뿐. 객관적 재무 근거 없음 | Opex 15% 이상 절감 가능한 ROI 수치 확보 후 즉각 승인 |
| **Adjacent-02** | 클레어 리 (구매본부장, 39세) | 확장 | 해외 원청사가 공급망 데이터 입증 요구. 생산 이력 데이터가 없어 거래 위협받음 | 생산 데이터 투명한 이력 관리로 글로벌 공급망 실사 통과 |
| **Adjacent-03** | 김가이드 (경영 고문, 46세) | 확장 | AI 실패 사례가 워낙 많아 고객에게 섣불리 추천 못함. 추천 후 실패 시 평판 파괴 우려 | 신뢰할 수 있는 AI 벤더 파트너 확보. 고객 성공률 유지 |
| **Extreme-01** | 노태풍 (창업주, 61세) | 극단 | '내 30년 직관을 기계가 대체한다'는 분노. 시스템 도입 시 프로젝트 자체를 방해함 | (암묵적) 자신의 현장 노하우와 경험이 회사에 유산으로 남기를 원함 |
| **Extreme-02** | 차품질 (품질이사, 51세) | 극단 | AI의 확률적 판단이 불량을 야기할 것이라는 극단적 불신. 0.001% 오차도 용납 불가 | 불량 발생의 '사전 이상 징후'를 과학적 데이터로 포착. 클레임 원인 분석 자동화 |
| **Non-user-01** | 최보안 (CISO, 43세) | 비활성 | 클라우드 AI는 데이터 유출 가능성이 있어 보안 정책상 도입 불가. 사고 시 본인 책임 | 클라우드 없이 사내에서 완결되는 보안 100% 준수 AI 아키텍처 확보 |
| **Non-user-02** | 한평안 (대표, 54세) | 비활성 | '지금 흑자인데 왜 건드리냐'는 현상 유지 집착. 변화 비용과 리스크 자체를 거부 | (현재 없음) 외부 압력(원청사 스마트화 요구 등) 발생 시 비로소 목표 생성 |

## 🎯 Pain/Goal 키워드 클러스터 분석

> 위 12인의 Pain과 Goal을 반복적으로 등장하는 키워드로 묶으면, 우리 솔루션이 해결해야 할 핵심 과제가 명확해진다.

### Pain 클러스터 (반복 키워드)

| 클러스터 | 해당 페르소나 | 핵심 공통 Pain |
| :--- | :--- | :--- |
| **데이터 활용 불가** | Core-01, 02, 03, 04 | 데이터는 쌓이는데 연동·분석·활용이 전무함 |
| **특정 인물 의존** | Core-02, 04 | 숙련자 또는 담당자 1인 부재 시 운영 마비 |
| **투자 실패 공포** | Core-05, Adj-01, Adj-03 | 투자 대비 성과 미입증 시 커리어·평판 손상 |
| **AI 신뢰 부재** | Ext-02, Non-01 | AI의 오류 가능성과 보안 사고 리스크에 대한 극단적 거부 |
| **변화 저항** | Ext-01, Non-02 | 현재 방식의 고수 또는 기술 자체에 대한 철학적 거부 |

### Goal 클러스터 (반복 키워드)

| 클러스터 | 해당 페르소나 | 핵심 공통 Goal |
| :--- | :--- | :--- |
| **시스템화(탈 인력 의존)** | Core-01, 02, 04 | 특정 사람 없이도 공장이 최적으로 돌아가는 구조 |
| **가시적 ROI 수치** | Core-05, Adj-01, Adj-03 | 도입 전·후 명확한 수치 비교가 필수 |
| **데이터 통합** | Core-03, Adj-02 | 파편화된 시스템을 연결하는 단일 진실의 원천(Single Source of Truth) 확보 |
| **보안 준수 혁신** | Non-01 | 보안 정책을 깨지 않으면서도 AI 혜택을 누리는 방법 |
| **조기 경보 시스템** | Core-02, Ext-02 | 문제가 터지기 전에 AI가 먼저 알려주는 예측 체계 |

### §5-2. 신뢰성 — 가용성 산출 공식 추가 `[V5.1 수정]`

| 항목 | 요구사항 |
|:---|:---|
| 시스템 가용성 | **≥ 99.5%** (월간, 계획 유지보수 제외) |
| **가용성 산출 공식** `[V5.1]` | `가용률(%) = (총 월간 시간 − 계획 유지보수 시간 − 비계획 중단 시간) / (총 월간 시간 − 계획 유지보수 시간) × 100`. 계획 유지보수 = **사전 72시간 고지**, 월 최대 **4시간** 이내. 비계획 중단 = 월 최대 **3.6시간** (총 720h 기준 99.5%) |
| AI 오류율 | STT 오인식 ≤ **10%**, Vision 실패 ≤ **15%** |
| 데이터 무결성 | 감사 리포트 불일치율 ≤ **1%** |
| 장애 복구 (RTO) | 원격: ≤ **4시간** / 현장 출동: 수도권 ≤ **4시간**, 비수도권 ≤ **8시간** |
| 데이터 백업 (RPO) | ≤ **1시간** (온프레미스 로컬 백업) |
| **MTBF** | **≥720시간 (30일)** — 계획되지 않은 시스템 중단 간격 |
| **MTTR** | **≤2시간** — 장애 감지→서비스 정상화 소요 시간 |
